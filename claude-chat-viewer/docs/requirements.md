# 需求文档 — Claude Chat Viewer

## 项目概述

一个 Windows 桌面应用，用于浏览和管理 Claude Code 的本地聊天记录。**只读访问** `~/.claude/` 目录下的转录文件，不修改任何 Claude Code 数据。支持自定义收藏夹，将不同项目中的会话归纳整理。

## 功能需求

### F1 — 项目与会话浏览
- 自动扫描 `~/.claude/projects/` 下所有项目的会话转录
- 左侧边栏以树形展示：项目 → 会话列表
- 会话标题优先显示 AI 自动生成的标题，无标题则显示第一条用户消息
- 显示每个会话的时间、消息数量
- 拼音/中文项目名友好显示

### F2 — 对话查看
- 主区域以聊天气泡形式展示完整对话
- 用户消息靠右、Claude 回复靠左
- Markdown 渲染（标题、列表、代码块、表格）
- 代码块语法高亮
- 支持虚拟滚动（处理大转录文件，最多 5000+ 条消息）

### F3 — Thinking 折叠
- Claude 的思考过程**默认折叠**，显示为缩略条
- 点击可展开查看完整思考内容
- 折叠状态本地记忆，不写入 .claude 目录

### F4 — 工具调用展示
- 工具调用以卡片形式展示（Bash、Read、Write、Grep、WebSearch 等）
- 按工具类别着色区分
- 显示工具输入参数，输出过长时默认折叠

### F5 — 全文搜索
- 支持按关键词搜索所有会话
- 搜索结果展示匹配片段和上下文
- 点击结果跳转到对应消息并高亮
- 快捷键 `Ctrl+K` 聚焦搜索框

### F6 — 会话元数据
- 显示会话基本信息：模型名称、时间范围、消息数
- 显示 Claude Code 版本、工作目录

### F7 — 自定义收藏夹（核心功能）
- 用户可创建自定义收藏夹（类似音乐播放器的"播放列表"）
- 将会话添加到收藏夹中（**不移动原文件**，仅创建引用）
- 同一会话可加入多个收藏夹
- 支持拖拽会话到收藏夹
- 收藏夹数据存储在 App 自己的数据目录（`%APPDATA%/claude-chat-viewer/`）
- 支持收藏夹重命名、删除（仅删除引用，不影响 Claude Code 原始数据）
- 侧边栏同时显示「按项目浏览」和「收藏夹」两种视图模式
- 支持给收藏夹内的会话添加备注/标签

### F8 — 会话导出
- 支持将单个会话导出为 Markdown 文件
- 可选择是否包含 Thinking 内容
- 导出到用户指定目录

## 非功能需求

- **只读安全**：任何代码路径不得写入 `~/.claude/` 目录
- **App 自有数据**：收藏夹、用户偏好等存储在独立目录，与 Claude Code 数据完全隔离
- **并发兼容**：Claude Code 同时运行时，共享读模式打开文件，不冲突
- **向前兼容**：Claude Code 更新时未知字段不报错，穿透保留
- **性能**：最大转录文件（~6MB，5000+ 行）加载时间 < 2 秒
- **编码**：UTF-8，处理中文字符不乱码
- **UI 风格**：浅色主题，参考 Claude 网页版 (claude.ai) 风格
