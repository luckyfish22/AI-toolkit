# 设计规范 — Claude Chat Viewer

## 设计原则

参考 **Claude 网页版 (claude.ai)** 风格：浅色主题、温暖友好、干净简洁、橙色调点缀、低视觉噪音。

## 配色方案（浅色主题）

### 主色调

| 用途 | 色值 | 说明 |
|------|------|------|
| 页面背景 | `#fafafa` | 最外层背景 |
| 主内容区背景 | `#ffffff` | 聊天区、卡片 |
| 侧边栏背景 | `#f5f5f5` | 左侧面板 |
| 悬浮（Hover） | `#f0f0f0` | 列表项悬浮 |

| 文字（主） | `#1a1a1a` | 正文标题 |
| 文字（次） | `#666666` | 时间戳、元数据 |
| 文字（淡化） | `#999999` | 占位文本、辅助说明 |

## 布局

```
┌──────────┬──────────────────────────────┐
│ TopBar   │ 搜索栏 + 标题                 │
│ 56px     │ 浅灰底 + 底部边框             │
├──────────┼──────────────────────────────┤
│ Sidebar  │ ChatView                      │
│ 280px    │ 自适应                         │
│ 浅灰背景 │ 白色背景                       │
│          │                               │
│ ▸ 视图切换│ ┌─────────────────────────┐  │
│   ○ 项目 │ │ 消息气泡                  │  │
│   ○ 收藏 │ │                           │  │
│ ──────── │ │                           │  │
│ 项目树   │ │                           │  │
│ 或收藏夹 │ └─────────────────────────┘  │
└──────────┴──────────────────────────────┘
```

- Sidebar 可折叠（快捷键 `Ctrl+B`）
- 侧边栏顶部有视图切换：**按项目浏览** / **收藏夹**
- 收藏夹视图中每个收藏夹是独立分组
- 最小窗口尺寸：900×600
- 默认窗口尺寸：1200×800

## 组件设计

### Sidebar
- 浅灰背景 `#f5f5f5`
- 视图切换：两个 Tab（项目/收藏夹），Claude 橙下划线激活态
- 项目项：文件夹图标 + 项目名，点击展开会话列表
- 会话项：聊天气泡图标 + 标题，显示日期，hover 时出现「添加到收藏夹」按钮
- 收藏夹项：星形图标 + 收藏夹名，点击展开内部会话列表

### 收藏夹管理
- 右键会话 →「添加到收藏夹」→ 弹出收藏夹选择菜单
- 底部有「+ 新建收藏夹」选项
- 收藏夹视图中，每个收藏夹右侧有「…」菜单（重命名/删除）
- 收藏夹内会话可右键移除

### ChatView
- 白色背景 `#ffffff`
- 消息气泡样式：
  - 用户气泡：浅蓝底 + 蓝灰细边框，无圆角尖角（圆形气泡）
  - Claude 气泡：白底 + 浅灰边框，圆形气泡
- 消息间距：24px
- 日期分隔线：居中灰色文字 + 两侧横线

### Thinking 块
- 浅橙背景 `#fef3c7`，左边框橙色 `#d97706`
- 折叠时显示「 Thinking」（橙色文字）
- 展开后等宽字体深灰文字

### 工具调用卡片
- 浅灰背景 `#f9fafb`，左边框着色
- 头部：工具图标 + 工具名（加粗）
- 输入区：代码块展示
- 输出区：默认折叠，点击展开

### 搜索
- TopBar 中置搜索框，圆角，浅灰背景
- `Ctrl+K` 聚焦
- 搜索结果面板从侧边栏滑入，覆盖项目列表
- 结果项显示：会话标题、匹配片段（关键词高亮为 Claude 橙背景）

### 空状态
- 居中显示
- 图标 + 引导文字："选择一个会话开始浏览"
- 或 "没有匹配的搜索结果"
- 文字颜色 `#999999`

## 交互规范

- 过渡动画：150ms ease-in-out
- Hover 状态：背景色变化
- 收藏夹拖拽：拖拽时显示虚线插入指示
- Toast 提示：「已添加到收藏夹」「收藏夹已删除」等操作反馈
